##Deployment instructions

```
$ heroku create speedcoder
$ heroku buildpacks:set heroku/ruby
$ heroku buildpacks:add --index 1 heroku/nodejs
$ git push heroku master
$ heroku run rails db:migrate
$ heroku run rails db:seed
```
